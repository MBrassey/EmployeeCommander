## EmployeeSQL

Manage Departments, Employees & Roles with this node backend application and MySQL database.

#### Requirements

    node
    npm

#### Installation

    npm i
    npm run migrate

#### Usage

    Usage: node server.js [ -h | -v | -l | -a ]

    [options]

           -h          Display this message.
           -v          Show version.
           -l          Show license info.
           -a          What is EmployeeSQL?
